# Certifications: AZ-900 Microsoft Azure Fundamentals

Study notes, explainers, and walkthroughs for the **Microsoft AZ-900: Azure Fundamentals** certification.

AZ-900 covers cloud concepts, core Azure services, and Azure pricing, support, and governance. While the focus of this channel is Power Platform, understanding Azure is increasingly important for Power Platform developers, especially for integration scenarios covered in PL-400.

## Who this is for

- Anyone preparing for the AZ-900 exam
- Power Platform developers wanting a broader understanding of the Microsoft cloud
- Career switchers and beginners new to cloud
- Anyone working alongside Azure-based teams

## What this repo will cover

- Cloud concepts: IaaS, PaaS, SaaS, public, private, hybrid
- Core Azure services: compute, networking, storage, databases
- Azure identity, governance, and compliance
- Azure cost management and SLAs
- Hands-on walkthroughs of key services relevant to Power Platform developers

Several of the projects in [certifications-pl400](https://github.com/HeatherAmi/certifications-pl400) use Azure services (Function Apps, Service Bus, webhooks). A working knowledge of Azure fundamentals makes those projects much easier to follow.
